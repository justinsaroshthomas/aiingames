The Minimax Algorithm

Understanding the decision-making engine behind the unbeatable AI

🌳

Game Tree Exploration

Minimax recursively builds a complete game tree, exploring every possible sequence of moves from the current state to all terminal states (win, loss, or draw).

📊

Score Evaluation

Terminal states receive scores: +10 for AI wins, -10 for human wins, 0 for draws. Depth is factored in to prefer faster wins.

🔄

Maximize / Minimize

The AI maximizes its score on its turn and assumes the opponent will minimize it. This guarantees optimal play against any strategy.

✂️

Alpha-Beta Pruning

Eliminates branches that can't affect the final decision, reducing computation by up to 50% while producing identical results.

minimax_algorithm.py 
def minimax(game, depth, is_maximizing, alpha, beta):
    # Base case: evaluate terminal states
    if game.check_winner() == ai_player:
        return 10 - depth     # AI wins (prefer faster)
    if game.check_winner() == human:
        return -10 + depth    # Human wins
    if game.is_draw():
        return 0              # Draw

    if is_maximizing:
        best = -∞
        for move in available_moves:
            score = minimax(game, depth+1, False, alpha, beta)
            best = max(best, score)
            alpha = max(alpha, score)
            if beta <= alpha: break  # ✂️ Prune!
        return best
    else:
        best = +∞
        for move in available_moves:
            score = minimax(game, depth+1, True, alpha, beta)
            best = min(best, score)
            beta = min(beta, score)
            if beta <= alpha: break  # ✂️ Prune!
        return best
